KeyLogger

Instructions for installing the KeyLogger VS Code extension

  1. Download the extension from here.
  2. Open VS Code.
  3. Open View > Extensions.
  4. Press the "..." button in the top-right corner of the Extensions view.
  5. Select "Install from VSIX...". (See screenshot below.)
  6. Choose the KeyLogger.vsix file.
  7. After installation, you should see a message in the bottom right corner stating, "Keylogger is activated!".
  8. After editing a source code file, you should see a file called "HackUSU_Keylogs.db" appear on your desktop. This is the log file.

To uninstall the extension:

  1. Open VS Code.
  2. Open View > Extensions.
  3. In the search bar of the Extensions view, type "KeyLogger".
  4. Right-click on the KeyLogger extension and select "Uninstall".
  5. Restart VS Code.
