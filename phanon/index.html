Phanon

Instructions for trying out Phanon as an instructor

  1. Go to https://phanon.academy
  2. Create an account (you don't have to use your real email address)
  3. Click the + New Course button to add a course
  4. Click Copy Course
  5. Input the code 88f3345d-4809-48b6-8d36-8d687573495a for a demo Python course
  6. Click Copy from Share Code
  7. Select the course once it has finished importing
  8. Click the People tab to invite students and see the list of students in the class
    • If you want to test the course as a student, invite a student using another email and then create an account using that email address
  9. Click on a student to see which lessons they have completed
    • You can look a student's progress on a lesson by clicking the lesson
  10. Click the dropdown next to a lesson and select edit to edit the lesson

Instructions for trying out Phanon as a student

You will need to have created an instructor account and invited a student if you would like test the tool as a student. Note: normally students may not proceed to the next exercise or lesson until the current exercise is complete. This behavior is disabled in the demo so that you may freely try exercises in different lessons.
  1. Go to https://phanon.academy
  2. Create an account
  3. If the instructor has invited you to the course then you will see the course invite at the top of the page
  4. Click Accept on the invite
  5. Click on the new course
  6. Select one of the lessons to begin
